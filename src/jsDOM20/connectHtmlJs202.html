JavaScript and the DOM Connecting HTML and JavaScript overview: learn how to add script elements to HTML and connect external JavaScript files to HTML pages. JavaScript brings the life to the page. Terms: Document Object Model: DOM, the data representation of the HTML elements that make up the structure of a web page, serving as away for JavaScript to understand the structure of an HTML page in order to make it dynamic. child element: an element that's nested inside another element parent element: an element that has one or more elements nested inside it The script tag To include some JavaScript on a page, add an HTML script tag to the BOTTOM of the element. The script tag takes an attribute named src; this attribute specifies the JavaScript file to be included. Here's an example: If your browser happens across a script tag when it's rendering a requested web page, it will request the file and execute it immediately.
